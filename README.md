# Vinyl Record Store: Laravel, Vue and AJAX Build 🎵💿
<img src="frontend/images/vinyl-readme.jpg" alt="readme vynil image" width="800">

This project is an assignment for the Multimedia Authoring 4 (MMED-5006) and Multimedia & The Internet (MMED-3014) courses in the Interactive Media Design program at Fanshawe College (Winter 2026).

The goal of this project is to develop a web application that displays a collection of vinyl records using a Vue.js frontend connected to a Laravel API backend. Users can browse records, filter content, and view detailed information dynamically without reloading the page.

## Installation 💻

1. Click on the repository link 
2. Click the green "< > Code" button
3. Select "Download ZIP"
4. Extract the folder to your computer


## Laravel Setup ⚙️

1. Download both the frontend and Laravel backend folders
2. Move them inside your MAMP or WAMP `htdocs` directory
3. Check that the `.env` file matches your local database credentials and ports
4. Verify the fetch URL inside `main.js` matches your local server configuration
5. Create a database inside phpMyAdmin
6. Import the provided SQL file
7. Run MAMP or WAMP
8. Open the project in your browser using localhost


## Usage 🚀

1. Download project folders
2. Move them into your `htdocs` directory
3. Confirm database credentials inside `.env`
4. Confirm fetch endpoints inside `main.js`
5. Start MAMP or WAMP
6. Open project through localhost
7. Browse vinyl records and click individual items to view details dynamically


## Project Overview 🎶

This application simulates a digital vinyl record store where users can:

- Browse a collection of vinyl albums
- View album details dynamically
- Filter records by artist, genre, or title
- Interact with content without page reloads
- Fetch data from a custom-built Laravel API

The interface is responsive and works from mobile to desktop screens.


## Assignment Requirements 📋

- Create a Vue.js application using a single HTML file and Vue instance
- Connect the frontend to a custom Laravel API
- Display a sorted listing of vinyl records
- Allow users to click records to view album details
- Load record details dynamically without navigating to another page
- Implement loading indicators and error handling
- Ensure responsive layout across devices
- Optional enhancements using GSAP animations
